# CalculatorApp C# WinForms

Проект демонстрирует принципы ООП, обработку исключений, использование паттернов Стратегия и Фабрика, а также алгоритм сортировочной станции для вычисления выражений.

## Возможности

- **Сложение, вычитание, умножение, деление**
- **Алгоритм сортировочной станции** для разбора и вычисления выражений
- **Учитывание приоритета знака при вычислении**
- **Паттерн Стратегия** для выделения отдельных операций как стратегий
- **Паттерн Фабрика** для получения нужной стратегии по символу оператора
- **Обработка исключений**:
  - Деление на ноль
  - Несоответствующие скобки
  - Неверный формат или неизвестные токены

## Использование

1. Введите арифметическое выражение в текстовое поле (например: `2 + 2 * 2`).
2. Нажмите кнопку **Вычислить**.
3. Результат отобразится справа от кнопки.
4. В случае ошибки появится окно сообщением об ошибке.
